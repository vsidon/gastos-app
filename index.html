🔗 Conectar a Google Sheets
Sigue los pasos del instructivo para obtener tu URL de Google Apps Script. Una vez configurado, cada gasto se guarda automáticamente en tu hoja de cálculo.
Sin configurar
📋 Instructivo Google Sheets
1. Ve a sheets.new y crea una hoja llamada Gastos

2. En la fila 1 pon estos encabezados exactos:
Fecha | Hora | Categoria | Monto | Nota | Semana | Mes

3. Ve a Extensiones → Apps Script

4. Borra el código y pega el código de abajo

5. Clic en Implementar → Nueva implementación → App web

6. Acceso: Cualquier persona → Implementar → copia la URL

7. Pega esa URL arriba y guarda
📝 Código Apps Script
function doPost(e) { try { var data = JSON.parse(e.postData.contents); var ss = SpreadsheetApp.getActiveSpreadsheet(); var sheet = ss.getSheetByName('Gastos') || ss.getActiveSheet(); var now = new Date(); var week = getWeekNumber(now); sheet.appendRow([ Utilities.formatDate(now, 'America/Tijuana', 'yyyy-MM-dd'), Utilities.formatDate(now, 'America/Tijuana', 'HH:mm:ss'), data.categoria || '', parseFloat(data.monto) || 0, data.nota || '', week, Utilities.formatDate(now, 'America/Tijuana', 'yyyy-MM') ]); return ContentService .createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); } catch(err) { return ContentService .createTextOutput(JSON.stringify({status:'error',msg:err.toString()})) .setMimeType(ContentService.MimeType.JSON); } } function getWeekNumber(d) { var d2 = new Date(Date.UTC(d.getFullYear(), d.getMonth(), d.getDate())); var dayNum = d2.getUTCDay() || 7; d2.setUTCDate(d2.getUTCDate() + 4 - dayNum); var yearStart = new Date(Date.UTC(d2.getUTCFullYear(),0,1)); return Math.ceil((((d2 - yearStart) / 86400000) + 1)/7); } function doGet(e) { return ContentService .createTextOutput(JSON.stringify({status:'ok',msg:'GastoRapido activo'})) .setMimeType(ContentService.MimeType.JSON); }
